5. BMI Calculator with Accuracy 📏⚖️
Purpose: Provide accurate BMI calculation and health category assessment with detailed insights.
Implementation Details:

BMI Calculation Formula:

BMI = weight(kg) / (height(m))²
Support both metric (kg, cm) and imperial (lbs, inches) units
Auto-conversion between units


Input Fields:

Height (cm or ft/inches with toggle)
Weight (kg or lbs with toggle)
Age (for context)
Gender (for body composition insights)


BMI Categories & Health Status:

Underweight (BMI < 18.5) - Blue
Normal Weight (BMI 18.5-24.9) - Green
Overweight (BMI 25-29.9) - Yellow
Obese Class I (BMI 30-34.9) - Orange
Obese Class II (BMI 35-39.9) - Red
Obese Class III (BMI ≥ 40) - Dark Red


Additional Metrics Calculated:

Ideal weight range (based on height)
Weight difference from ideal (kg needed to gain/lose)
BMI trend (if historical data available)
Body composition estimate (considering age/gender)


Accuracy Features:

Real-time calculation as user inputs data
Visual feedback with color-coded status
Comparison with WHO standards
Age and gender-adjusted interpretation


Health Recommendations Based on BMI:

Dietary suggestions
Exercise recommendations
Health risks associated with current BMI
Target weight and healthy habits


UI Components:

Interactive BMI Calculator card
Real-time result display with gauge/meter
Category badge with color coding
Health insights panel
Historical tracking (show previous measurements)
Printable BMI report


Integration with Diet System: Pass BMI data to AI Diet Recommendation for personalized suggestions
